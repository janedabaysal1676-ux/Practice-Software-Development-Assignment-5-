<h1 align="center"> Maze Game</h1>


# Game Rules

- The game world is a 2D discrete grid. 
- The maze size should be configurable 
- The player starts at position (0, 0).
- The player and the guard cannot leave the grid. 
- Random cells contain walls that block movement. That is, the player and the 
guard cannot move through walls. 
- The guard moves randomly within the grid, after the player moves. 
- Random cells contain treasures. 
- The player wins by collecting all treasures. 
- The player loses if he is caught by the guard. 

# Controls

- W → Up
- S → Down
- A → Left
- D → Right

# Required Classes
- Entity
- Player
- Guard
- Map
